Apresentação

O BlueMoon é uma API back-end desenvolvida em C#/.NET para oferecer um gerenciamento completo e eficiente de lojas de informática. Inicialmente criado para a Atuall Informática, seu design permite expansão para outras empresas. O sistema facilita o controle de produtos, clientes, usuários e vendas, além de estar preparado para incorporar futuramente o gerenciamento de fornecedores e ordens de serviço. Seu objetivo principal é automatizar e otimizar processos internos, tornando a gestão mais eficiente, segura e confiável, e garantindo crescimento junto com o negócio.
